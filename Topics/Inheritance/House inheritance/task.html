House inheritance

There is a program that reads two numbers from the input, each in a new line:

  1. The first one represents the number of rooms in the house. It helps define and create an instance of a class depending on that number according to the following rules: 1 for Cabin, 2-3 for Bungalow, 4 for Cottage, 5-7 for Mansion, 8-10 for Palace. If the number is lower than 1, it's a Cabin. If the number is greater than 10, it's a Palace.
  2. The second one is the base price of this house. If this price is lower than 0, this house costs 0, if it is greater than 1_000_000, the base price is 1_000_000.

There is also a coefficient per each class: 1 for Cabin, 1.2 for Bungalow, 1.25 for Cottage, 1.4 for Mansion, 1.6 for Palace.

The program should define which house needs to be built and calculate the final price of this house according to the next rule: base price * coefficient.

Fix the program below and implement the given logic.

Don't forget to print the integer, not a floating-point number!


Sample Input:
-5
12000

Sample Output:
12000



Memory limit: 256 MB
Time limit: 8 seconds
