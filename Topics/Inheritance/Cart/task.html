Cart

There is a program that reads two values from the input, each in a new line:

  1. The first one represents the product type as a string: headphones, smartphone, tv, laptop. All in lowercase.
  2. The second one is the price of the product. It must be from 0 to 1000000.

The program must calculate the total price of a product considering the product tax: for headphones, it's 11%, for a smartphone, it is 15%, for tv, it is 17%, and for a laptop, it is 19%.

Total price = base price + base price * product tax

Fix the program below and implement the given logic.


Sample Input:
headphones
200

Sample Output:
222



Memory limit: 256 MB
Time limit: 8 seconds
